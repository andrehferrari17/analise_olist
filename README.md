# 📊 Análise de Dados E-commerce OLIST

Projeto de análise de dados utilizando a base pública da OLIST com foco em modelagem relacional, consultas SQL analíticas, engenharia de dados e visualização de métricas de negócio.

## 🚀 Objetivo do Projeto

O objetivo deste projeto é simular um ambiente real de análise de dados de um marketplace, explorando:

* Modelagem relacional no PostgreSQL
* Criação de tabelas, constraints e relacionamentos
* Tratamento e padronização de dados
* Consultas SQL analíticas
* Construção de métricas de negócio
* Criação de views analíticas para BI
* Exploração de hipóteses de negócio utilizando dados reais

---

# 🛠️ Tecnologias Utilizadas

* PostgreSQL
* SQL
* Git & GitHub
* Power BI
* Python (futuramente para análises avançadas)

---

# 🧩 Modelagem dos Dados

O banco foi estruturado utilizando o modelo relacional da base OLIST, contendo tabelas como:

* customers
* orders
* order_items
* order_payments
* order_reviews
* products
* sellers
* geolocation

Foram implementados:

* Primary Keys
* Foreign Keys
* Índices
* Ajuste de tipos de dados
* Relacionamentos entre entidades

# 📈 Análises Desenvolvidas

## 🌎 Análise Geográfica

* Estados e cidades com maior volume de vendas
* Concentração geográfica de vendedores
* Distância entre vendedor e cliente
* Impacto logístico regional

## 🚚 Análise Logística

* Tempo médio de entrega
* Atrasos por estado
* Impacto da distância no prazo de entrega
* Relação entre atraso e avaliação do cliente

## 🛒 Análise de Produtos

* Categorias mais vendidas
* Impacto de fotos e descrição nas vendas
* Relação entre peso/frete e prazo de entrega

## 💳 Análise de Pagamentos

* Métodos de pagamento mais utilizados
* Parcelamento por categoria e região
* Ticket médio por estado

## ⭐ Análise de Satisfação

* Score médio por estado
* Relação entre atraso e review score
* Sellers com melhores avaliações
